Introduction to Power Automate Desktop

  • Overview of Robotic Process Automation (RPA) and its benefits
  • Introduction to Power Automate Desktop: features and capabilities

Installation & Setup
  • System requirements for installing Power Automate Desktop

  • Configuration and setup post-installation
First Desktop Flow
  • Understanding flow design concepts
  • Adding actions to create a basic automation flow

Core Concepts: Variables, Conditions, and Loops

Variable Actions
  • Set Variable: assigning values to variables
  • Truncate Number: adjust Number precision
  • Get Random Numbers: generate Random values
  • Increase/Decrease Variable: adjust numeric values
  • Create New List: initialize a List Variable
  • Add/Remove Item From List: manipulate List contents
Conditional Actions
  • If and Else: basic branching logic
  • Else If: additional conditions
  • Switch and Case: multi-branch decision-making
  • If File/Folder Exists: check File or Folder presence
Loop Operations
  • For Loop: iterate a fixed Number of times
  • While Loop: repeat until a condition is met
  • Do-While Loop: execute at least once
  • Break and Continue: control Loop execution flow
3

Text, Date, File & Excel Automation

Text Processing
  • Find/Replace/Extract/Concatenate/Split Text: manipulate text
Date and Time Operations
  • Get Current Date and Time: retrieve system time
  • Format Date and Time: adjust date/time display
  • Convert Time Zones: manage time zone conversions
File Management
  • Create/Read/Write/Append to File: basic file operations
  • Copy/Move/Rename/Delete File: manage file system
  • Zip Files/Unzip Files: archive and extract file collections
Excel Automation & Recording
  • Read Excel File: retrieve data From Excel
  • Write/Append to Excel: update or Add data
  • Find and Replace: search and modify data in Excel
  • Record/Play/Pause Recorded Web and Desktop Actions
4

Web, UI, Outlook & PDF Automation

Web Automation
  • Launch Web browser
  • Click Element: interact with web elements
  • Type into Field: input data
  • Extract Data from Web: scrape web content
  • UI elements & Selectors with Fallback features
UI Automation
  • Click/Input Text/Extract Text/Wait for Element
  • Automate desktop application interactions
  • UI elements & Selectors with Fallback features
Email & Document Processing
  • Create/Send Outlook Email: automate email composition
  • Save Outlook Email Attachments: archive email data
  • Read PDF Text: extract text content from PDFs
  • Extract Data from PDF: parse structured data
Database Operations
  • Connect to Database
  • Execute SQL Query
  • Fetch/Update/Insert/Delete Data
5

Advanced Topics: AI, Cloud & Work Queues

Cloud Integration
  • List and Create files in OneDrive
  • Download OneDrive files into local drive
AI Builder & Copilot
  • Explore GPT inbuilt actions
  • Introduction to Copilot: leveraging AI for automation
  • Benefits and limitations of AI-driven task automation
Work Queue Management
  • Implementing work queues: managing task assignments
  • Task prioritization: optimizing workflow efficiency

Enterprise Deployment & Monitoring

Machine Management
  • Registering machines for automation: setting up machines for RPA tasks
  • Managing machine pools: distributing workload across registered machines
  • Attended vs Unattended Bots: differences and deployment considerations

  • Overview of Cloud Flows: benefits of cloud-based automation
